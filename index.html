Connectez l'app à Firebase pour synchroniser le stock en temps réel sur tous les appareils.

📋 Étapes (5 min) :

2 → Créer un projet (ex : chok-beton)
3 → Cliquer </> Ajouter une app Web, donnez-lui un nom
4 → Menu gauche → Firestore Database → Créer → Mode Test
5 → Revenez sur l'app Web → Paramètres du projet → copiez l'objet firebaseConfig
